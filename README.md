# Discord-Faramir

A server monitoring bot for discord, specifically made for Faramir, the mistreated (and possibly fire-prone) server.

## Features

- **!sensors** - Outputs average and highest CPU temps.
- Overheat checker
- Scheduled shutdown notifications
